Guide de Vérification des Meta Tags

Meta Tags Essentiels

  • charset - Encodage des caractères
  • viewport - Configuration pour appareils mobiles
  • X-UA-Compatible - Compatibilité IE

Meta Tags SEO

  • description - Description de la page
  • keywords - Mots-clés (moins important aujourd'hui)
  • robots - Instructions pour les robots
  • author - Auteur du contenu

Meta Tags Réseaux Sociaux

  • og:title - Titre Open Graph
  • og:description - Description Open Graph
  • og:type - Type de contenu
  • twitter:card - Type de carte Twitter

Meta Tags Mobile

  • theme-color - Couleur du thème mobile
  • apple-mobile-web-app-capable - Capacité application mobile
  • apple-mobile-web-app-status-bar-style - Style de la barre de statut

Points à Vérifier

  • Présence des meta tags essentiels
  • Configuration correcte du viewport
  • Description pertinente pour le SEO
  • Configuration appropriée pour les réseaux sociaux
  • Gestion du cache si nécessaire
  • Support mobile approprié
